# Tourism in Crete - Travel Guide

A simple static website showcasing travel information about Crete, including accommodation, activities, and sightseeing, using **HTML** and **CSS**.

## Featuring
- Overview of sights to see
- Accommodation recommendations
- Plethora of activities and experiences
- Sources and contact page

### Note
This website was developed as an assignment for the first-semester course "Introduction to Computer Science".\
My chosen topic was "a Travel Guide presenting a specific destination".
